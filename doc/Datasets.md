# Dataset 網站整理
以下為整理之大資料集的網站，幫助有興趣做分析的快速找到想要分析的資料集：

* [Yahoo! Datasets](http://webscope.sandbox.yahoo.com/catalog.php?datatype=g)
* [Big Data Bench](http://prof.ict.ac.cn/BigDataBench/)
* [Quora Datasets](http://www.quora.com/Where-can-I-find-large-datasets-open-to-the-public)
* [Wiki Dump Database](http://dumps.wikimedia.org/backup-index.html)
* [Social Event Radar](http://api.ser.ideas.iii.org.tw/docs/#!/fb_fanpage_search)
* [NASA OpenData](https://data.nasa.gov/developer)
* [OpenData Taiwan](http://wiki.opendata.tw/doku.php)
* [53.5 billion clicks dataset](http://www.bigdatanews.com/profiles/blogs/53-5-billion-clicks-dataset-available-for-benchmarking-and-testin)
* [Big Data Ecosystem](https://github.com/zenkay/bigdata-ecosystem)
* [Aws Datasets](http://www.kdnuggets.com/2015/04/awesome-public-datasets-github.html)
